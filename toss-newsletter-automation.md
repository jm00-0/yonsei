# 토스증권 코스피 급등락 뉴스레터 자동화 작업 정리

작성일: 2026-07-02

## 최종 자동화 설정

- 자동화 이름: 토스증권 코스피 급등락 종목 뉴스레터
- 실행 주기: 주중 월요일부터 금요일까지 매일 오후 4시
- 시작 기준: 2026-07-02 오후 4시, Asia/Seoul 기준
- 발송 방식: Gmail 이메일 발송
- 뉴스레터 언어: 한국어
- 대상 데이터: 토스증권에서 확인 가능한 KOSPI 보통주

## 뉴스레터 선정 기준

매일 토스증권 기준으로 최신 거래일 KOSPI 보통주 중 하루 등락률이 다음 조건에 해당하는 종목을 찾는다.

- 상승률 +5% 이상
- 하락률 -5% 이하
- 조건에 해당하는 종목이 10개 이상이면 절대 등락률이 큰 순서로 10개 선정
- 조건에 해당하는 종목이 10개 미만이면 가능한 만큼만 작성하고 그 사실을 명시

제외 대상:

- 코스닥 종목
- ETF, ETN
- 우선주
- 스팩
- 단일종목 ETP
- 기타 특수상품

## 주요 데이터 소스

토스증권 공개 웹/API를 우선 데이터 소스로 사용한다.

- 토스증권 국내 랭킹
- 토스증권 종목 상세 정보
- 토스증권 뉴스 검색 및 뉴스 상세
- 토스증권 종목 댓글

랭킹 조회 조건 예시:

- 급상승: `heavy_soar`
- 급하락: `heavy_descent`
- 기간: `duration=1d`
- 국가 태그: `tag=kr`

## 종목별 포함 내용

각 종목마다 아래 내용을 포함하도록 구성했다.

- 종목명
- 종목코드
- 상승 또는 하락 방향
- 등락률
- 기준 시각
- 토스증권 기준 가격
- 거래대금 또는 섹터/TICS 정보
- 등락 원인 요약
- 관련 뉴스 기사 링크 1~2개
- 토스증권 댓글 반응 요약

등락 원인은 토스증권 뉴스, 공시성 기사, 테마, 수급, 댓글 분위기를 중심으로 정리한다. 직접 원인 기사가 확인되지 않는 경우에는 추정임을 명시한다.

## 댓글 반응 요약 방식

토스증권 댓글은 원문을 길게 인용하지 않고 분위기를 요약하는 방식으로 정리한다.

예시 분류:

- 추가 상승 기대
- 손절 또는 물림 반응
- 차익실현 분위기
- 테마 기대
- 불안 또는 의심
- 정책/산업 이슈 논쟁

## 이메일 형식

이메일 제목:

```text

```

본문 구성:

1. 데이터 기준 시각
2. 한줄 요약
3. 종목별 번호 목록
4. 공통 테마와 주의할 점
5. 출처와 방법
6. 투자 유의 문구

하단 문구:


## GitHub 업로드 참고 절차

현재 작업 폴더 자체에는 별도 Git 저장소가 설정되어 있지 않다. GitHub에 올리려면 현재 폴더에서 새 저장소를 초기화하거나, 기존 GitHub 저장소에 이 파일을 복사해서 커밋하면 된다.

현재 폴더를 새 Git 저장소로 올리는 예시:

```bash
cd /path/to/current/workspace
git init
git add toss-newsletter-automation.md
git commit -m "docs: add Toss newsletter automation summary"
git branch -M main
git remote add origin https://github.com/YOUR_ID/YOUR_REPO.git
git push -u origin main
```

기존 저장소가 있다면 `git remote add origin` 대신 기존 원격 저장소 설정을 사용하면 된다.
