# 데이터 수집 방식

이 프로젝트는 매일 KOSPI 등락률 상위 종목을 찾은 뒤 뉴스레터를 보냅니다.

## 기본 흐름

1. 먼저 `pykrx`로 KOSPI 전체 종목의 등락률을 조회합니다.
2. `pykrx` 조회가 실패하면 네이버증권 상승/하락 종목 페이지로 자동 대체합니다.
3. 절댓값 기준 등락률 5% 이상 종목 중 상위 10개를 고릅니다.
4. 고른 종목의 네이버증권 뉴스, 토론, 리서치 후보 링크를 수집합니다.

## pykrx 환경변수

`pykrx`의 일부 KRX 데이터 조회는 KRX 로그인이 필요할 수 있습니다.

필요한 경우 GitHub Secrets에 아래 값을 추가할 수 있습니다.

| Secret 이름 | 설명 | 필수 여부 |
| --- | --- | --- |
| `KRX_ID` | KRX 로그인 ID | 선택 |
| `KRX_PW` | KRX 로그인 비밀번호 | 선택 |

이 값이 없거나 `pykrx`가 실패하면 네이버증권 대체 수집으로 넘어갑니다.

## 네이버증권 대체 수집

대체 수집은 아래 공개 페이지를 사용합니다.

- KOSPI 상승 종목: `https://finance.naver.com/sise/sise_rise.naver?sosok=0`
- KOSPI 하락 종목: `https://finance.naver.com/sise/sise_fall.naver?sosok=0`

이 방식은 로그인 없이 동작합니다.

단, 과거 날짜 조회가 아니라 실행 시점의 현재 시세 페이지를 기준으로 합니다. GitHub Actions가 매일 16:00 KST에 실행되므로 일일 뉴스레터 용도에는 이 대체 경로를 사용할 수 있습니다.

## 공개 저장소 주의사항

`KRX_ID`, `KRX_PW`, Gmail 앱 비밀번호는 절대 코드나 문서에 직접 쓰지 마세요.

민감한 값은 GitHub Secrets에만 저장해야 합니다.
