# Project Guidelines

이 문서는 Yonsei Stock Report Mailer 팀의 공통 작업 기준입니다.

## 1. 서비스 목표

매일 오후 16:00 KST에 국내 KOSPI 종목 중 등락률이 큰 종목을 골라, 관련 리서치, 댓글, 뉴스를 요약한 뉴스레터를 Gmail로 보냅니다.

## 2. 종목 선정 기준

- 대상 시장은 국내 KOSPI입니다.
- 전일 종가 대비 등락률의 절댓값이 5% 이상인 종목만 포함합니다.
- 조건을 만족한 종목 중 등락률 절댓값이 큰 순서로 상위 10개를 선택합니다.
- 상승과 하락을 모두 포함합니다.
- 거래일이 아니거나 조건을 만족하는 종목이 없으면 메일에 그 사실을 적습니다.

## 3. 요약 기준

각 종목은 아래 세 영역으로 나누어 요약합니다.

- 리서치: 증권사 분석 리포트나 투자 의견 관련 내용
- 댓글: 투자자 반응, 여론, 반복적으로 보이는 의견
- 뉴스: 공시, 실적, 산업 이슈, 정책, 사건 등 뉴스성 내용

각 영역은 3줄로 요약합니다.

- 각 줄은 짧고 사실 중심으로 작성합니다.
- 투자 추천, 매수 권유, 확정적 수익 표현은 쓰지 않습니다.
- 원문 전체를 복사하지 않습니다.
- 출처 링크를 함께 남깁니다.

## 4. 마크다운 저장 기준

결과물은 날짜별 폴더에 저장합니다.

```text
reports/
  YYYY-MM-DD/
    index.md
    005930-samsung-electronics.md
    000660-sk-hynix.md
```

`index.md`에는 그날의 전체 요약과 종목별 상세 파일 링크를 넣습니다.

종목별 파일에는 아래 순서를 사용합니다.

```markdown
# 종목명 (종목코드)

- 날짜: YYYY-MM-DD
- 등락률: +5.8%
- 출처: 네이버증권 / 토스증권 / KB증권

## 리서치 요약

1. ...
2. ...
3. ...

## 댓글 요약

1. ...
2. ...
3. ...

## 뉴스 요약

1. ...
2. ...
3. ...

## 참고 링크

- [뉴스 제목](https://example.com)
```

## 5. 메일 발송 기준

- 발송 시간은 매일 오후 16:00 KST입니다.
- GitHub Actions 기준 cron 시간은 UTC 07:00입니다.
- 메일 제목 형식은 `[Yonsei Stock Report] YYYY-MM-DD KOSPI 급등락 요약`입니다.
- 메일 본문에는 상위 10개 종목의 핵심 요약과 상세 마크다운 링크를 넣습니다.
- 수신자는 환경 변수나 GitHub Secrets로 관리합니다.

## 6. 비밀값 관리

공개 저장소이므로 아래 값은 절대 커밋하지 않습니다.

- Gmail 주소
- Gmail 앱 비밀번호
- API 키
- 수신자 이메일 목록
- 개인 계정 정보

필요한 값은 GitHub repository secrets에 저장합니다.

예상 secret 이름은 아래처럼 맞춥니다.

```text
GMAIL_SENDER
GMAIL_APP_PASSWORD
NEWSLETTER_RECIPIENTS
```

## 7. 브랜치 작업 기준

`main`은 팀 공통 기준과 안정 버전만 둡니다.

기능 개발은 아래 브랜치에서 진행합니다.

- `feature/toss-securities-reports`: 토스증권 담당
- `feature/kb-securities-reports`: KB증권 담당
- `feature/naver-securities-reports`: 네이버증권 담당

작업이 끝나면 Pull Request를 열어 `main`으로 합칩니다.

## 8. 네이버증권 담당 범위

네이버증권 브랜치에서는 아래 작업을 우선 구현합니다.

1. KOSPI 등락률 5% 이상 종목 찾기
2. 등락률 절댓값 기준 상위 10개 고르기
3. 종목별 네이버증권 뉴스와 댓글 수집하기
4. 리서치, 댓글, 뉴스 영역으로 나누어 각 3줄 요약하기
5. 날짜별 마크다운 파일 만들기
6. 뉴스레터 본문 만들기
7. 매일 16:00 KST 자동 실행 준비하기

## 9. 테스트 기준

새 기능을 만들 때는 테스트도 같이 추가합니다.

예상 테스트 대상은 아래와 같습니다.

- 등락률 필터링
- 상위 10개 정렬
- 요약 결과 형식
- 마크다운 파일 생성
- 메일 본문 생성

테스트 실행 명령은 아래로 통일합니다.

```bash
pytest
```
